# Progression Annuelle — Informatique Cycle 3

_Année scolaire 2025-2026 – École Saint-Anne_
_Préparé par Jean-Noël Schilling_
_Références : Programme officiel cycle 3 – BOEN n°31 du 30 juillet 2020 + mise à jour 2023_

## Introduction

- **Public** : 21 élèves de CM1/CM2
- **Rythme** : 45 min/semaine + 3 grands travaux pratiques (4 à 6 séances chacun)
- **Objectif général** : Développer les compétences du domaine « Informatique et création numérique » en s'appuyant sur des projets motivants et concrets, tout en consolidant les gestes de base (organisation, recherche, sauvegarde, usage raisonné de l'IA).
- **Matériel** : 1 poste par élève, accès internet filtré, ENT, suite LibreOffice, Scratch 3, Canva Éducation, Audacity.

## Lien avec les attendus de fin de cycle 3 (BOEN 2023)

- Programmer des déplacements et des actions simples → TP1 (Scratch)
- Concevoir un programme qui répond à un cahier des charges → TP1 + TP2 + TP3
- Décomposer une instruction complexe en instructions élémentaires → activités débranchées + tous les TP
- Représenter des données (images, sons, textes) → TP1 (sons/images), TP2 (graphiques), TP3 (audio)
- Rechercher et organiser des informations sur internet de façon critique → TP2
- Produire et diffuser un document numérique → les 3 TP
- Comprendre le fonctionnement d'un ordinateur (matériel/logiciel) → variante Mai (démontage)

## Calendrier

- **Septembre-Octobre** : Bases + algorithmes débranchés (ex. : "Ping Pong, Robot obéissant, découvrir le nombre caché").
- **Novembre** : Initiation programmation (Scratch avancé : boucles, variables).
- **Décembre** : TP1 (Cartes Noël).
- **Janvier-Février** : Représentation données (codage images, graphiques simples).
- **Mars-Avril** : TP2 (Journal printemps).
- **Mai** : Audio/collaboration (bases podcast) ou séance technologique sur l'exploration des composants d'ordinateur.
- **Juin** : TP3 (Histoire collective) + bilan.
